Introdução ao JavaScript

Bem-vindo à nossa introdução ao JavaScript!

O que é JavaScript?

JavaScript é uma linguagem de programação utilizada principalmente para adicionar interatividade e dinamismo às páginas da web. Com ele, os desenvolvedores podem manipular elementos da página, responder a eventos do usuário e criar aplicativos web avançados.

Por que aprender JavaScript?

JavaScript é uma das linguagens mais populares para o desenvolvimento web. Ele é usado para adicionar interatividade, validação de formulários, animações e muito mais aos sites.

Exemplos de Código JavaScript

Exemplo 1: Variáveis

var nome = "João";
var idade = 30;
console.log(nome + " tem " + idade + " anos.");

Exemplo 2: Funções

function saudacao() {
  return "Olá, mundo!";
}
console.log(saudacao());

Exemplo 3: Estruturas de Controle

var hora = 10;

if (hora < 12) {
  console.log("Bom dia!");
} else {
  console.log("Boa tarde!");
}

Exemplo 4: Loops

for (var i = 0; i < 5; i++) {
  console.log("Número: " + i);
}
